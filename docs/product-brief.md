# Product brief

## Problema
Merchants with multiple locales lack a focused workbench for missing, outdated and visible translations.

## Usuario objetivo
International growth manager or small merchant selling into multiple languages/markets.

## Merchant persona
Merchant con catalogo activo, equipo chico y necesidad de resolver un problema operativo puntual sin instalar una suite pesada.

## Flujo principal
Choose locale, scan visible resources, review missing fields first, export gaps and plan updates.

## Propuesta de valor
Translation completeness dashboard without becoming a generic machine translation app.

## MVP
- Locale coverage dashboard by resource type.
- Gap table for missing, stale and low-priority fields.
- Settings for locales, markets and resource priority.
- Explicit no-AI/no-auto-translation v1 stance.

## Futuro fuera de alcance
- CSV export and import review.
- Optional machine translation draft generation.
- Market-specific coverage.
- Translation digest validation before writes.
